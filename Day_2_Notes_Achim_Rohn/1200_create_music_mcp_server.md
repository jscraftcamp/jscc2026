# Create Music MCP Server

The project is Open Source: https://music.build/

It allows you to give the AI agent the tools to create music by creating Midi files. The tools are exposed via an MCP server.

So the agent provides the musical theory, like notes, what instrument, and with the help of the MCP server it can convert to a midi file or sheet music.

In essence the LLM generates a musical "program" that also be iterated upon just like writing code.

Also mentioned was https://opendaw.org/ as a possible way to integrate this functionality directly inside there, so that it is easy to work on.
