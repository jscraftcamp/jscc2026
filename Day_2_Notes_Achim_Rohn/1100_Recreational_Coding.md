# Recreational Coding

The idea here is to rediscover the joy of coding manually. It is best done by doing one achievable task in something that interests you. The more obscure the better. It should be just for fun and not produce any value.

You don't use the AI to build it, but using it as a tutor or sparring partner is OK.

One benefit of this is preventing AI burnout, but you also keep your developer skills sharp, because they can atrophy pretty quickly.
