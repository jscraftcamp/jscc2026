# The state of PWA

Through extended browser capabilities, PWAs now feature an impressive list of capabilities that can them make very close to native apps.

But the support for the is still limited in some places, notably on iOS. Which still today makes it hard to truly use them everywhere. 

The consensus is, that depending on your usecase they can be a powerful alternative to native apps, because they have some advantages:

* not being tied to an app store, needing to pass reviews
* new versions of the app are available instantly everywhere
