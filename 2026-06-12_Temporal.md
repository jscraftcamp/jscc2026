# Temporal

Session hosted by [Jannes](https://www.linkedin.com/in/jbantje/) and [Marco](https://www.linkedin.com/in/stnimmerlein/).


We started with some questions from [jsdate.wtf](https://jsdate.wtf).

Then we used some [slides](https://m7.codes/slides/temporal/TemporalEN.pdf) from a talk, but we deviated a lot from it and discussed about the need of precise, localized times (why only Instant is not enough) and tried out some special cases.

For example: `Temporal.ZonedDateTime.from({year: 2026, month: 1, day: 31, timeZone: Temporal.Now.timeZoneId()}).add({months: 1})` results in 2026-02-28 (so remaining days of the month get truncated).
